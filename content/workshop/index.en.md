
# NutriTrack — Full-Stack AWS Deployment Workshop

#### Overview

This workshop provides a complete, step-by-step guide for deploying **NutriTrack** — a full-stack, serverless nutrition tracking platform on AWS. You will build a React Native (Expo) mobile application backed by **AWS Amplify Gen 2**, **Amazon Bedrock AI** (Qwen3-VL), **AppSync GraphQL**, **DynamoDB**, and a containerized **FastAPI** backend on **ECS Fargate**. The system uses AI to analyze food photos, process voice recordings, and provide personalized nutrition coaching.
