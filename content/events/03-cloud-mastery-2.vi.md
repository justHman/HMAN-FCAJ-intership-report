# AWS Cloud Mastery 2 - Cloud Engineering & IaC

**Ngày:** 4 tháng 4, 2026
**Địa điểm:** Hội trường A - Đại học FPT TP.HCM, TP. Hồ Chí Minh
**Vai trò:** Người tham dự (FCJ Cloud Intern - Team NeuraX)

## Mô tả sự kiện

Buổi meetup chuyên sâu về kỹ thuật cloud và phát triển phần mềm cho cộng đồng FCJ. Chương trình tập trung vào bài toán triển khai hệ thống, tối ưu hiệu năng bằng ngôn ngữ thế hệ mới và tự động hóa hạ tầng bằng công cụ IaC.

## Các hoạt động chính

Ba phiên trình bày chính:
**Phiên 1: Container Orchestration & Kubernetes** - Lộ trình từ Docker/Compose đến Kubernetes, cấu trúc Control Plane, Worker Node, Pod, ConfigMap, Secret và so sánh Amazon EKS với Self hosted Kubernetes.
**Phiên 2: Lập trình hàm với Elixir & Ứng dụng IoT** - Trình bày sức mạnh concurrency và fault tolerance của BEAM, kèm demo dự án Smart Greenhouse dùng MQTT.
**Phiên 3: Infrastructure as Code (IaC) Tools** - So sánh CloudFormation, CDK, Terraform và demo triển khai S3 static web bằng Terraform + LocalStack.

## Kết quả

- Hiểu rõ hơn kiến trúc Kubernetes và khi nào nên chuyển từ Docker sang K8s.
- Nắm được triết lý "let it crash" và sức mạnh supervisor tree của Elixir cho backend IoT ổn định.
- Phân biệt rõ vai trò của CloudFormation, CDK và Terraform.
- Củng cố quy trình IaC an toàn: plan trước apply, không commit state/secret và tách module để tái sử dụng.
