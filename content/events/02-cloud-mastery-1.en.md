# AWS Cloud Mastery 1 - AI and Serverless Practice

**Date:** March 14, 2026
**Location:** AWS Vietnam Office (Floor 26), Ho Chi Minh City
**Role:** Attendee (FCJ Cloud Intern - Team NeuraX)

## Event Description

A deep dive knowledge sharing session for Vietnam's tech community, connecting developers and cloud enthusiasts to explore applied AI and AWS services through practice driven, real world projects.

## Main Activities

Three focused technical sessions:
**Session 1: Building AI Agents & Frameworks** - Presented an open SDK that automates tool definitions (web search, Python execution) and integrates with Amazon Bedrock; emphasized memory, multi step reasoning, and tool calling.
**Session 2: Prompt Engineering & Serverless Architecture** - Covered role, chain, and tree of thought prompting, and demoed a "Prompt Optimizer" extension built entirely on AWS Serverless services (S3, CloudFront, Cognito, API Gateway, Lambda, DynamoDB, Bedrock).
**Session 3: Applying AI to Real Products** - Showcased a smart cabinet IoT project using Raspberry Pi, IoT Core, S3, DynamoDB, Rekognition, plus a personal finance app using Textract to extract receipt data by context.

## Outcomes

- Agentic AI is a clear trend: agents can automate complex workflows through iterative reasoning and tool use.
- Prompt engineering reduces cost; structured prompts and English inputs lower token spend.
- Serverless stacks deliver scalable backends with minimal operations overhead.
- Amazon Rekognition is strong for vision tasks, and Custom Labels enable domain specific training.
- Amazon Textract adds context awareness for receipts and complex documents where OCR falls short.
